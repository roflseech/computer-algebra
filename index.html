Computer algebra

Some computer algebra methods that work with long numbers.
Supports Natural numbers, Integers, Rational numbers and Polynomials.
Each operation is built using previously defined operations, you can check source codes to see the algorithms, and see which other operations they use.

To do any operation - input the arguments, separating them with commas, and press operation buttton. Result will appear in bottom text field.
Operations are performed relatively first argument. For example, for input "21, 7" - Minus command's result will be 14.

For fraction input use / to separate numerator from denominator. For example "5/6". For input "2/3, 5/2" multiplication result will be "5/3". Also, if denominator is equal to 1, you can skip it - "5" is equivalent to "5/1"

Input example for polynomials:
1/2x^2 + 2x - 4
x^2 - x^4 + 5/3x^120 - 222222/777777
More info:
Space and power signs are not necessary("x^2 - 2" equals to "x2-2")
No coefficient is being interpreted as coefficient equal to 1("1x" is equal to "x")
No power sets power equal to 1("x^1" is equal to "x")
No Ñ… means that power is 0("2x^0" is equal to "2")
