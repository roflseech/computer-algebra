Компьютерная алгебра

Для того чтобы выполнить какое-либо действие - достаточно ввести аргументы(разделяя их запятыми) и нажать на нужную кнопку, после чего в самом нижнем текстовом поле появится результат
Действие будет рассматриваться относительно первого аргумента. Например при вводе "21, 7" и нажатии на "Вычесть" результат будет 14.

При вводе дробей нужно использовать / для отделения числителя от знаменателя. Например 5/6. При вводе 2/3, 5/2 и нажатии на "Умножить" результат будет 5/3. Также если знаменатель равен нулю, то можно просто написать "5", что будет эквивалентно "5/1"

Пример ввода многочленов:
1/2x^2 + 2x - 4
x^2 - x^4 + 5/3x^120 - 222222/777777
Подробнее:
Пробелы и знаки степени вводить не обязательно(т.е. x^2 - 2 = x2-2)
Остутствие коэффициента = коэффициент равен 1(т.е. 1x = x)
Отсутствие степени = степень равна 1(т.е. x^1 = x)
Отсутствие х = степень равна 0(т.е. 2x^0 = 2)
